# Algebra Data Structures Project

## Description
This project contains solutions for:
- Data structures
- Vector operations
- Matrix operations
- Tensor operations

## Files
- scalar.py → basic variables
- norms_and_unit_vectors.py → unit vector and norm
- matrix_tensors.py → matrix operations
- generic_tensor_notation.py → tensor explanation
- vectors_and_vector_transposition.py → transpose

## Tools Used
- Python
- NumPy
- TensorFlow
- PyTorch
